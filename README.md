## Introduction

Dashboard to launch github workflows

## Getting Started

Env file:
```yaml
GITHUB_OWNER=<your github username>
GITHUB_REPO=<your github repo>
GITHUB_WORKFLOW_ID=<your workflow name>
GITHUB_TOKEN=<your github token>
```
